Contenedor Desplazable

Desplázate horizontal y verticalmente usando las barras de scroll

Contenido Desplazable

Sección 1: Introducción

Este es un contenedor con barras de scroll horizontales y verticales. El contenedor tiene un ancho de 600px y una altura de 400px, pero el contenido es más grande, lo que genera las barras de desplazamiento.

Utiliza la propiedad CSS overflow: auto para mostrar las barras de scroll solo cuando sea necesario.

Sección 2: Contenido Horizontal

El contenido interno tiene un ancho de 1000px, lo que es mayor que el ancho del contenedor (600px). Esto genera una barra de scroll horizontal.

También hay suficiente contenido vertical para generar una barra de scroll vertical.

Sección 3: Más Contenido

Puedes desplazarte en ambas direcciones para ver todo el contenido. Las barras de scroll aparecen automáticamente cuando el contenido excede las dimensiones del contenedor.

Sección 4: Ejemplo Práctico

Este tipo de contenedores desplazables son útiles para mostrar tablas grandes, listas extensas o cualquier contenido que exceda el espacio disponible en la pantalla.

Los usuarios pueden navegar fácilmente a través del contenido sin necesidad de JavaScript.

Sección 5: Propiedades de Overflow

overflow: auto - Muestra barras de scroll solo cuando sea necesario.

overflow: scroll - Siempre muestra las barras, incluso si no hay contenido adicional.

overflow: hidden - Oculta el contenido que excede el contenedor.

overflow: visible - Muestra el contenido fuera del contenedor.

Sección 6: Final

Este es el final del contenido. Como puedes ver, el contenedor se desplaza tanto horizontal como verticalmente para acomodarse al contenido más grande.
